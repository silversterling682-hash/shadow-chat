# 🟢 Shadow WhatsApp - React & Python Project

A beautiful WhatsApp clone built with **React** frontend and **Python Flask** backend.
Features real-time messaging, encryption, and auto-delete messages!

---

## ✨ Features

- 🎨 **Exact WhatsApp Design** - Looks exactly like the original
- 📱 **Responsive Layout** - Sidebar + Chat area
- 🔐 **Secure Login System** - Register and Login
- 🛡️ **End-to-End Encryption** - Messages are encrypted
- ⏱️ **Auto Delete Messages** - Set timer (5s, 10s, 30s, 1m)
- 🔄 **Real-time Updates** - No page refresh needed
- 📱 **Mobile Friendly**

---

## 🛠️ Tech Stack

- **Frontend:** React (CDN version)
- **Backend:** Python Flask
- **Database:** JSON File
- **Security:** CryptoJS for encryption
